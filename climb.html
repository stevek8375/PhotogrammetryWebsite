Climbing App

This is an exploration of how create an app to help climbers along specific routes.

  • Get Started

Photogrammetry

Using photogrammetry, all of the images from my photoshoot produced a digital replica of the boulder face containing the pictographs.

Phone use consideration

Since viewing the sites through a smart phone restricts the field of view, no effort was needed to make a complete 360 environment.

Enhancement

Some pictographs can be very difficult to see with the naked eye. There is an image enhancer called DStretch that was created specifically to make pictographs more legible. Using this as a reference, I created a version that was enhanced without altering the surrounding rock surface.

AR function

Within the app, users can not only jump into different sites, but can also slide the viewing mode from seeing the pictographs as in real life to a better view of all of the details.
